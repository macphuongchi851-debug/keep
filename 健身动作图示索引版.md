# 健身动作图示索引版

> 版本：v1（先做稳定可用的索引版）
> 
> 说明：
> - 这版先给你做成 **动作图示/动图索引**，优先放 **稳定的动作页链接**。
> - 有些动作图站会做反爬，若首选页打不开，就直接点 **Bing 图片** 或 **B站视频** 备用链接。
> - 后面如果浏览器抓图环境更顺，再升级成 **真正内嵌图片/动图版**。

---

## 使用方法

1. 先看 **首选图示页**：适合快速看动作标准轨迹。  
2. 如果首选页打不开：点 **Bing 图片** 看图示。  
3. 如果还想看节奏和动作细节：点 **B站视频**。  
4. 训练当天只看对应动作，不用全翻。
