# MEMORY.md

> **用途**：当前主助手的长期记忆总索引。
> **什么时候写**：当出现当前主助手需要长期保留的稳定偏好、执行纪律、专属约定，或需要登记新的专题文件时。
> **写什么**：主助手专属的重要长期记忆，以及共享记忆入口。
> **不写什么**：其他 bot 的人格设定、其他 bot 的 daily 过程、明明适合放进 shared 的公共内容细节。

用于长期记忆（索引版）。这里只保留主助手专属核心记忆，并链接到公共记忆层。

## 主助手专属

- `openclaw-tutor` 每日课程默认发完整版讲义，至少包含：学什么、为什么、具体步骤、示例 / 操作思路、验收标准、常见坑、做完后怎么回报；不能只发摘要提纲。

## 共享记忆入口

- 跨 bot 公共记忆：`shared/MEMORY_GLOBAL.md`
- 通用环境 / 排障经验：`shared/environment-global.md`
- 通用术语 / 约定：`shared/terminology.md`

## 主助手本地专题索引

- 项目经验：`memory/project-experience.md`
- 环境 / 排障经验（本地扩展）：`memory/environment-notes.md`
- 沟通规矩（主助手专属）：`memory/rules.md`
