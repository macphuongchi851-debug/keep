第 0 步：安装前准备

  • 一台 x86 电脑 / 小主机 / NAS 主机
  • 至少 1 块硬盘，推荐用 SSD 做系统盘
  • U 盘（8GB 以上）
  • 显示器、键盘、网线
注意：飞牛官方明确说明，安装时需要接网线；当前不支持只靠无线网卡安装。

第 1 步：下载飞牛 OS 镜像

打开飞牛官方下载页，选择 fnOS。页面上会看到“复制 fnOS 下载链接”“如何安装 fnOS？”等按钮。

如果你懒得网页里点，直接下载官方 ISO 即可：

https://iso.liveupdate.fnnas.com/x86_64/trim/fnos-1.1.23-1627.iso

图 1：飞牛官方下载页。这里能看到 fnOS 下载入口、ARM 设备列表、安装教程入口。

第 2 步：下载 Rufus 并制作启动 U 盘

飞牛官方文档推荐使用 Rufus 或 Balena Etcher。Windows 用户优先用 Rufus，最省事。

官方给出的 Rufus 下载地址：

https://static2.fnnas.com/installer/rufus-4.5.exe

图 2：Rufus 官方页面。下载后打开，选择 U 盘、选择 fnOS 的 ISO，然后点击“开始”。
  • 设备：选择你的 U 盘
  • 启动类型：选择刚下载的 .iso 文件
  • 分区类型：优先 GPT；老机器不认再试 MBR
  • 点击开始，等待状态变成“准备就绪”

第 3 步：给待安装机器接线

  • 接好系统盘（推荐 SSD）
  • 接好网线
  • 接好电源、显示器、键盘
  • 插上刚做好的启动 U 盘

第 4 步：开机进入启动菜单

开机时快速按这些键之一：F2、F7、F8、F10、F11、F12、Esc、Delete。

进入 Boot Menu 后，选择你的 U 盘启动。

第 5 步：进入安装器并选择安装方式

进入安装页面后，优先选择 Graphical Install（图形化安装）。

如果报错：
1）若提示需要关闭 Secure Boot，就去 BIOS 里把 Secure Boot 关掉；
2）如果图形化安装失败，重启后改选 Rescuing Install（应急模式安装）。

第 6 步：选择系统盘

只选择那块准备用来装系统的硬盘。通常是容量较小的 SSD（如 128GB / 256GB）。

高危提醒：被选中的系统盘会被格式化，里面的内容会被清空。别把装数据的硬盘选进去。

第 7 步：设置系统分区和 Swap

官方建议系统分区设置为 64GB。Swap 按安装器推荐值即可。

小白就按默认推荐来，先装成功再说，别上来瞎折腾。

第 8 步：确认格式化并开始安装

确认后，系统开始写入 fnOS。中途别断电、别拔盘、别手欠。

第 9 步：安装完成后设置网络

安装完成后，系统会自动检测网卡并尝试通过 DHCP 获取 IP。记住这个 IP 地址。

比如：192.168.1.20

第 10 步：拔掉 U 盘并重启

保存网络设置后，先把启动 U 盘拔掉，再重启系统。不然它容易重新进安装器。

第 11 步：浏览器访问飞牛 OS

在另一台电脑浏览器中打开上一步的 IP 地址。首次进入时，你会看到初始化页面。

图 3：飞牛官方帮助中心安装文档。里面写明了安装步骤、Graphical Install、系统分区建议 64GB、网络设置等。

如果裸 IP 打不开，再试：

  • http://你的IP:5666
  • https://你的IP:5667

第 12 步：完成初始化

  • 设置设备名称：支持数字、英文、短横线 -
  • 设置管理员账号：第一位管理员是超级管理员，密码一定要强
  • 进入系统后尽快创建存储空间

装完后建议立刻做的事

  • 确认系统盘和数据盘识别正常
  • 创建存储空间
  • 记录 IP 地址和管理员密码
  • 后续再慢慢折腾共享、影视库、Docker、远程访问
