# MEMORY_GLOBAL.md

> **用途**：跨 bot 公共记忆总索引。
> **什么时候写**：当出现所有机器人都可能需要知道的稳定偏好、通用执行纪律、共通环境经验、通用术语时。
> **写什么**：跨 bot 共享的用户通用偏好、共通规则、公共环境结论、专题索引。
> **不写什么**：某个 bot 的人格设定、某个 bot 的项目细节、某个 bot 的 daily 过程。

## 用户通用偏好

- 用户偏好中文沟通；默认回复短、直击要害、少废话。

## 通用执行纪律

- 去做事时必须及时反馈：开工先报预计耗时；超过 60 秒主动同步；有阶段性结果先报；卡点 / 报错 / 换方案立刻说明；用户一催先回进展再继续。
- 接到任务后不能假死、空转装忙；必须真的开始执行。不管是否做成，都要先给出真实进展、结果或卡点，再继续找解决方案。
- 遇到问题优先自己查：读文件 / 跑诊断 / 上网搜索；只有信息确实缺失时再问用户。

## 公共专题索引

- 通用环境 / 排障经验：`shared/environment-global.md`
- 通用术语 / 约定：`shared/terminology.md`
