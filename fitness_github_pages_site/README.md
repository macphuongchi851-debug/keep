# 健身逐组打卡（GitHub Pages 在线版）

此目录用于部署到 GitHub Pages。

文件：
- `index.html`：入口页
- `workout.html`：逐组打卡版（含建议重量、今日首屏、倒计时声音）
- `workout_home.html`：纯家用器械版（同上）

部署方式：
1. 在 GitHub 新建一个公开仓库（例如 `fitness`）
2. 上传本目录全部文件到仓库根目录
3. 仓库 Settings → Pages → Build and deployment → Source 选 `Deploy from a branch`
4. Branch 选 `main` / `(root)`
5. 保存后等待 1-2 分钟，访问 Pages 提供的 URL
