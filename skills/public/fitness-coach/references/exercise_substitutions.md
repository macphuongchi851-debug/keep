# Exercise substitutions (quick mapping)

## Squat pattern
- Back squat ↔ front squat ↔ safety-bar ↔ leg press ↔ goblet squat
- Knee pain: reduce depth/stance tweaks, try leg press or split squat

## Hinge pattern
- Deadlift ↔ trap-bar ↔ RDL ↔ hip hinge DB ↔ back extension
- Low back sensitive: trap-bar, RDL lighter, hip thrust, machines

## Horizontal push
- Bench press ↔ DB bench ↔ push-ups ↔ machine press
- Shoulder cranky: DB neutral grip, floor press, reduce ROM

## Vertical push
- OHP ↔ DB shoulder press ↔ landmine press
- Shoulder cranky: landmine press, higher reps, less load

## Horizontal pull
- Barbell row ↔ chest-supported row ↔ cable row ↔ DB row

## Vertical pull
- Pull-ups ↔ pulldown ↔ assisted pull-ups

## Single-leg
- Bulgarian split squat ↔ step-up ↔ reverse lunge ↔ leg extension (less ideal but OK)

## Glutes/hamstrings
- Hip thrust ↔ glute bridge ↔ cable pull-through
- Ham curl ↔ nordic (scaled) ↔ sliders

## Calves
- Standing calf raise ↔ seated calf raise

## Core (anti-extension/anti-rotation)
- Plank ↔ dead bug ↔ ab wheel (progression)
- Pallof press ↔ suitcase carry
