# Weekly check-in template

Ask the user to answer in bullets (copy/paste):

1) Goal right now (fat loss / strength / muscle / health):
2) Training last week (days completed / missed, key lift numbers):
3) Bodyweight (7-day avg + trend vs last week):
4) Steps (avg):
5) Sleep (avg hours + quality 1–10):
6) Stress (1–10):
7) Hunger (1–10):
8) Soreness / pain hotspots:
9) Adherence (diet/training) %:
10) Anything coming up next week (travel, social events):

Adjustment cheatsheet:
- Weight not moving (2+ weeks) + adherence good → -100~200 kcal or +1–2k steps.
- Performance dropping + sleep/stress poor → deload or reduce deficit/cardio.
- Joints angry → swap exercises (see substitutions) + reduce ROM/load.
