# Program templates (copy/paste)

## 3-day full body (beginner friendly)

**Rules**
- Main lift: 3–4 sets × 5–8 reps @RPE 6–8
- Accessories: 2–3 sets × 8–15 @RPE 7–9
- Progression: double progression (+1 rep/week until top of range, then +2.5–5% load)

### Day A
- Squat: Back squat / goblet squat
- Push: Bench press / push-ups
- Pull: Lat pulldown / pull-ups / row
- Hinge accessory: RDL / hip hinge DB
- Core: plank variation

### Day B
- Hinge: Deadlift / trap-bar / RDL
- Push: Overhead press / incline DB
- Pull: row variation
- Single-leg: split squat / step-up
- Carry: farmer carry

### Day C
- Squat variation: front squat / leg press
- Push variation: incline/close-grip
- Pull variation: chest-supported row
- Posterior chain: hip thrust / leg curl
- Core: anti-rotation (Pallof)

## 4-day upper / lower (intermediate default)

**Upper 1 (bench bias)**
- Bench press: 1×5 @RPE 8 + 3×5 @-10% load
- Row: 4×6–10
- Incline DB press: 3×8–12
- Lat pulldown/pull-ups: 3×8–12
- Lateral raise: 3×12–20
- Triceps + biceps: 2–3×10–15 each

**Lower 1 (squat bias)**
- Squat: 1×5 @RPE 8 + 3×5 back-off
- RDL: 3×6–10
- Split squat: 3×8–12/side
- Calves: 3×10–15
- Core: 3 sets

**Upper 2 (overhead/pull bias)**
- Overhead press: 1×5 @RPE 8 + 3×5 back-off
- Pull-up / pulldown: 4×6–10
- Incline bench: 3×6–10
- Chest-supported row: 3×8–12
- Rear delt: 3×12–20
- Arms: 2–3×10–15 each

**Lower 2 (hinge bias)**
- Deadlift / trap-bar: 3×3–5 @RPE 7–8
- Leg press / front squat: 3×8–12
- Hamstring curl: 3×10–15
- Glute bridge/hip thrust: 2–3×8–12
- Core: 3 sets

## Deload (1 week)
- Keep intensity moderate (RPE 6–7)
- Reduce sets by ~30–50%
- Keep movement patterns the same
