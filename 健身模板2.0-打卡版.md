# 健身模板 2.0｜表格/清单版打卡 Markdown

> 适用对象：男 / 42 岁 / 172cm / 78kg / 有训练基础 / 家庭训练 / 每周 5 练 / 每次 30–45 分钟  
> 器械：杠铃（最高 80kg）、哑铃（单只最高 20.5kg）、卧推凳、引体向上器（计划购入）  
> 目标：增肌减脂一起做 + 提体能 + 改体态 + 打基础辅助人体旗帜

---

## 一、每周总览

| 星期 | 训练主题 | 时长 | 核心目标 | 可选内容 |
|---|---|---:|---|---|
| 周一 | 上肢力量 | 35–45 分钟 | 卧推、划船、推举、引体 | 15–20 分钟轻有氧 |
| 周二 | 下肢力量 + 核心 | 35–45 分钟 | 深蹲、RDL、分腿蹲、核心 | 无 |
| 周三 | 体能恢复 / 有氧 / 旗帜基础 | 30–40 分钟 | 提体能、促进恢复、侧向核心 | 稳态有氧或循环体能 |
| 周四 | 上肢增肌 + 体态 | 35–45 分钟 | 肩背打开、胸背饱满、上肢线条 | 15–20 分钟轻有氧 |
| 周五 | 下肢增肌 + 后链 + 核心 | 35–45 分钟 | 臀腿后链、体态底盘 | 无 |
| 周六 | 可选有氧 / 恢复 / 旗帜基础 | 20–40 分钟 | 恢复、体能、旗帜基础 | 稳态有氧优先 |
| 周日 | 休息 | - | 恢复、走路、早点睡 | 轻拉伸 |

## 二、固定热身模板

### 1）上肢日热身（周一 / 周四）

- [ ] 开合跳或原地快步 1 分钟
- [ ] 肩绕环 前20 + 后20
- [ ] 俯卧撑 10–15 次
- [ ] 空杠卧推 2 组
- [ ] 肩胛后缩/下沉控制 10–15 次
- [ ] 轻重量划船 1 组

### 2）下肢日热身（周二 / 周五）

- [ ] 原地快步或高抬腿 1 分钟
- [ ] 徒手深蹲 15–20 次
- [ ] 髋部绕环 10 次/侧
- [ ] 臀桥 15 次
- [ ] 空杠深蹲 2 组
- [ ] 轻重量 RDL 1 组

### 3）体能/恢复日热身（周三 / 周六）

- [ ] 快步走/开合跳 1 分钟
- [ ] 胸椎转动 10 次/侧
- [ ] 髋部动态拉伸 10 次/侧
- [ ] 平板支撑 20 秒
- [ ] 肩胛控制 10 次

## 四、有氧安排规则

| 方案 | 安排 | 说明 |

| 保守版 | 周三 1 次 | 本周累、恢复一般时用 |

| 积极版 | 周一后 + 周三 + 周六 | 状态很好时用 |

### 有氧强度标准
- [ ] 微喘
- [ ] 能说话
- [ ] 不影响第二天力量训练

### 这周若出现以下情况，减少 1 次有氧
- [ ] 腿很沉
- [ ] 睡眠变差
- [ ] 深蹲/卧推明显退步
- [ ] 早上疲劳感高
- [ ] 心烦、恢复差

## 五、进阶规则（打印脑子里）

### 1）主动作（4×5–6）
- [ ] 如果 4 组都做到 6 次 → 下周加重量
- [ ] 如果没全做到 6 次 → 保持重量
- [ ] 上肢加 1.25–2.5kg
- [ ] 下肢加 2.5–5kg

### 2）辅助动作（3×8–10）
- [ ] 如果做到 10/10/10 → 下次加重量
- [ ] 加完重量后从 8 次重新爬

### 3）高次数动作（3×12–15）
- [ ] 先做标准
- [ ] 再追次数
- [ ] 最后再加重量

### 4）引体向上进阶
- [ ] 先追总次数
- [ ] 再追标准度
- [ ] 最后再考虑负重

### 5）状态差时调整
- [ ] 主动作少 1 组
- [ ] 辅助动作减 1–2 个
- [ ] 有氧减 1 次
- [ ] 不完全停练

## 六、4 周递进模板

| 周次 | 目标 | 训练策略 | 有氧 |
|---|---|---|---|
| 第1周 | 适应 | 找重量、找节奏、别上头 | 1–2 次 |
| 第2周 | 推进 | 主动作多 1 次或小加重 | 2 次 |
| 第3周 | 强化 | 本轮最强一周，但不乱冲 | 2 次以内 |
| 第4周 | 卸载 | 重量降到 85–90%，总量减 30–40% | 1–2 次轻松做 |

## 八、饮食执行清单（简版）

### 每日纪律
- [ ] 蛋白质 140–160g
- [ ] 每餐优先吃蛋白质
- [ ] 训练前后有点碳水
- [ ] 晚上少零食少酒
- [ ] 水喝够

### 三餐大方向
| 餐次 | 重点 |
|---|---|
| 早餐 | 鸡蛋 + 牛奶/酸奶 + 主食 |
| 午餐 | 蛋白质 + 主食 + 蔬菜 |
| 晚餐 | 蛋白质优先，少油炸 |
| 加餐 | 酸奶 / 蛋白粉 / 水果 / 豆制品 |
